Special Character Add-on for NVDA

This add-on allows users to insert special characters using keyboard shortcuts.

Shortcut Keys

  • Shift + Windows + 0 → § (Section or category, such as § 12) ALT + 0167
  • Shift + Windows + 1 → • (Bullet) ALT + 0149
  • Shift + Windows + 2 → ° (Degree, such as 30°C, corner, etc.) ALT + 0176
  • Shift + Windows + 3 → × (Multiplication mark, used instead of "x") ALT + 0215
  • Shift + Windows + 4 → ÷ (Division mark, used in mathematics) ALT + 0247
  • Shift + Windows + 5 → ± (Plus-minus, used in math or science) ALT + 0177
  • Shift + Windows + 6 → ‰ (Per thousand, used in statistics) ALT + 0137
  • Shift + Windows + 7 → √ (Square root, used in mathematics) ALT + 251
  • Shift + Windows + 8 → ≤ (Less than or equal to) ALT + 243
  • Shift + Windows + 9 → ≥ (Greater than or equal to) ALT + 242

What is Special Character Add-on?

Special Character is a supplement that helps to print 10 special characters commonly used in Word, Excel, or reports. You can quickly insert 10 special characters by pressing the shortcut keys Windows + Shift + 0 - 9, instead of waiting for the Alt (Alt Code) commonly used in printing government documents, reports, or schedules.

Who can use this add-on?

This add-on is useful for:

  • Scientific reporters
  • Excel works that use formulas or math symbols
  • Government documents with references, laws, etc.
  • Presentations that require professional symbols
